For those aiming to drive smaller commercial vehicles not covered by [Class A](/cdl-types/class-a-cdl) or [B](/cdl-types/class-b-cdl), such as passenger vans or [hazardous material transports](/cdl-types/endorsements/hazmat), a Class C Commercial Driver's License (CDL) is your ticket. Let's gear up for your journey to obtaining a Class C CDL.

<Snippet file="class-c-diagram.mdx" />

## Quick Reference: Class C CDL Essentials

| **Aspect** | **Details** |
|------------|-------------|
| **License Type** | Class C CDL |
| **Vehicle Types** | Small passenger buses, delivery vans, HazMat transport |
| **Passenger Capacity** | 16 or more (including the driver) |
| **HazMat Transport** | Vehicles carrying hazardous materials requiring placards |
| **Key Skills** | Passenger safety management, hazardous materials handling |
| **Potential Endorsements** | Passenger (P), Hazardous Materials (H) |
| **Skills Test Components** | Vehicle inspection, basic control skills, road test |
| **Employment Opportunities** | Shuttle bus driver, delivery services, HazMat transport |

## Decoding the Class C CDL

<AccordionGroup>
  <Accordion title="Defining Class C CDL">
    A Class C CDL qualifies you to handle vehicles transporting 16+ passengers (driver included) or hazardous materials requiring placards.
  </Accordion>

  <Accordion title="Common Vehicles for Class C">
    This license covers small passenger buses, delivery vans, and vehicles for hazardous material transport not needing Class A or B licenses.
  </Accordion>
</AccordionGroup>

## Your Roadmap to a Class C CDL

<Steps>
  <Step title="Verify Your Eligibility">
    Check you meet your state's age, medical, and legal criteria for a Class C CDL.
  </Step>

  <Step title="Acquire a Learner's Permit">
    Pass the [general knowledge](/essentials/general-knowledge) test to obtain your [CDL learner's permit](/cdl-types/cdl-permit).
  </Step>

  <Step title="Engage in Specialized Training">
    While not always mandatory, specialized training is beneficial, particularly for hazardous material handling and passenger safety.
  </Step>

  <Step title="Pass the Skills Test">
    Complete the [skills test](/essentials/skills-test) in a Class C vehicle, including [pre-trip inspection](/essentials/pre-trip), control skills, and a driving test.
  </Step>
</Steps>

## Class C CDL FAQs

  <Accordion title="Career Opportunities with a Class C CDL">

Embarking on the Class C CDL path opens doors to specialized driving roles in passenger transport and delivery services. Thorough preparation and understanding of the vehicle-specific requirements will steer you towards a successful driving career. For more insights on Class C CDL training and practice tests, head to [Driving-Tests.org](https://driving-tests.org).
